Doctor's Portal

Project Description
This App is built by combination of client and Server side.

1. Patient can meet with doctor's by fixed appointment. Additionally, it has dashboard, in which user and doctor's data store.
2. Appontment date can be changed by users. On the other hand admin can set up doctor's date, add doctor, manage doctor.
3. Firebase Authentication used to allow user to access my app.
4. jwt token added to verify user and get access to use website from server.
